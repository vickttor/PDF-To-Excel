# PDF TO EXCEL CONVERTER

### Dependencies

- Python 3.12.3
- Pip
- pandas
- tabula

### How to Run

```bash
python -m venv venv
```

```bash
./venv/Scripts/activate  # Windows
source venv/bin/activate # MacOS or Linux
```

```bash
pip install -r ./requirements.txt
```

```bash
python ./main.py
```
